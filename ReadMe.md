#Lexicon

Overview

The word “Lexicon” means vocabulary of a person, language, or branch of knowledge. As a result this program was created to understand the depth of a language.
This program is an (English-English) internet dictionary application which enables a user to find meaning, antonym, and synonym and also find translation to the entered word. It also has a text to speech feature to hear a correct pronunciation for the entered word.

##Modules

1. PyDictionary
2. Tkinter
3. os
4. tkMessagebox
5. pyttsx3

##System Requirements

- Internet connection (LAN or WLAN), speakers.
